# Product Next

A Next.js e-commerce frontend for displaying products, managing cart, and checkout functionality.

## Features

- Product listing with images, colors, sizes, and quantity selection
- Cart management using Zustand with localStorage persistence
- Checkout functionality
- Responsive layout with Tailwind CSS
- Header and Footer components
- Clean and modern UI

## Technologies Used

- [Next.js 13](https://nextjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Zustand](https://zustand-demo.pmnd.rs/) for state management
- [Lucide Icons](https://lucide.dev/) for cart and UI icons

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/alsatarysamah/Order_api.git
cd product_next


npm install


npm run dev

Open http://localhost:3000
 with your browser to see the result.
```
